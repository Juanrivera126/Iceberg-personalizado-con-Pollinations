Herramienta de inteligencia artificial que genera páginas en formato de presentación iceberg a partir de un tema introducido por el usuario, incluyendo:
1. Selector de estilos de imagen (31 estilos)
2. Selector de modelos de generación de imagen (Z Image, Flux Context, Nano Banana, Seedream4 y Nano Banana Pro)
3. Selector de de Niveles (3 a 7)
4. Posibilidad de regenerar imágenes en cada apartado y regenerar el apartado completo
5. Descarga del iceberg en formato HTML con imágenes en formato Base64
